Acoording to variable class various types of graphics are suggested. You can select one or several of them (in this latter case the various types of representations will be superposed).

All graphics correspond to package ggplot2's "geom" functions:

  • geom_bar displays bars, with height proportional to the number of cases in each group.
  • geom_bin2d displays rectangular bins (heatmap of 2d bin counts).
  • geom_boxplot displays box and whiskers plot. The upper whisker extends from the hinge to the highest value that is within 1.5 * IQR of the hinge, where IQR is the inter-quartile range, or distance between the first and third quartiles. The lower whisker extends from the hinge to the lowest value within 1.5 * IQR of the hinge. Data beyond the end of the whiskers are outliers and plotted as points (as specified by Tukey).
  • geom_density displays a smooth kernel density estimate.
  • geom_density2d displays contours from a 2d density estimate (2D kernel density estimation using kde2d).
  • geom_freqpoly displays frequency polygons. Uses lines to display a 1d distribution by dividing into bins and counting the number of observations in each bin.
  • geom_histogram displays a histogram. Uses bars to display a 1d distribution by dividing into bins and counting the number of observations in each bin.
  • geom_hex displays hexagonal bins (heatmap of 2d bin counts).
  • geom_jitter displays points, jittered to reduce overplotting.
  • geom_point displays points, as for a scatterplot.
  • geom_rug displays marginal rug plots.
  • geom_smooth: displays smooth of y=f(x), and 95% confidence intervals around smooth. For datasets with n < 1000 the smoothing method is loess. For datasets with n ≥ 1000 or more observations the smoothing method is a generalized additive model (GAM).
  • geom_violin displays a violin plot.
